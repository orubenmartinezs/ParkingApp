# Archivos de Sonido Requeridos

Por favor, agregue los siguientes archivos MP3 a esta carpeta para habilitar los efectos de sonido:

- `entry.mp3`: Sonido al registrar una entrada.
- `exit.mp3`: Sonido al registrar una salida.
- `success.mp3`: Sonido de éxito (ej. caja registradora) para pagos.
- `error.mp3`: Sonido de error.
- `sync_start.mp3`: Sonido al iniciar sincronización (opcional).
- `sync_success.mp3`: Sonido al terminar sincronización exitosamente.
- `online.mp3`: Sonido al recuperar conexión.
- `offline.mp3`: Sonido al perder conexión.

Si no se agregan estos archivos, la aplicación funcionará pero solo vibrará (si el dispositivo lo soporta).
